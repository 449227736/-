# 套管交付与质量专项改善 — 问题异常跟踪表

## 运行方式

双击 `index.html` 用浏览器打开即可使用。

## 功能

- **新增记录**：13 字段表单，支持图片上传（多张，预览，拖拽删除）
- **数据列表**：分页展示、关键词搜索、按问题类型/状态/责任人/日期筛选
- **查看详情**：完整展示单条记录
- **编辑记录**：修改已有记录，可新增/删除/替换图片
- **删除记录**：二次确认弹窗，防止误删
- **导出 Excel**：真 .xlsx 格式，标题红字加粗、表头灰色加粗、黑色边框、行高 60、图片内嵌单元格（按比例缩放 max 110x82px）

## 技术栈

- 纯前端单页应用，无需服务器
- 数据存储在浏览器 localStorage
- ExcelJS CDN 生成真 .xlsx 文件
- 图片以 base64 存储在记录中

## 字段说明

严格对齐参考 Excel「套管交付与质量专项改善问题异常跟踪表」的 13 列结构：

| 序号 | 问题提出人 | 异常问题描述 | 问题类型 | 图片/视频 | 解决措施 | 交付物 | 责任人 | 协助人 | 计划完成时间 | 完成情况 | 实际完成时间 | 备注 |

## 注意事项

1. 数据存储在浏览器 localStorage，清理浏览器数据会导致数据丢失
2. 建议定期导出 Excel 备份
3. 后续可接入后端数据库（代码结构已预留扩展点）
